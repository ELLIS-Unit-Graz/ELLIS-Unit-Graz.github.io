# {% include icon.html icon="fa-solid fa-robot" %}Human-Centred Computing

In our unit the focus of Human-centred Computing lies on machine learning for recommender systems and information retrieval, integrated with data science, user modeling, HCI, and inclusive technologies. Research spans psychology-informed recommender systems, user behavior analysis (e.g., beyond-mainstream music listeners), privacy-preserving methods like robust meta matrix factorization, temporal dynamics of preferences, and applied techniques such as autoencoders for session-based recommendations, all aimed at developing more personalized, adaptive, and user-centric interactive systems.
